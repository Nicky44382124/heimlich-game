# 🏥 汉川市向日葵社工中心  海姆利克急救法 - 儿童学习游戏

一个帮助儿童学习海姆利克急救法（Heimlich Maneuver）的互动小游戏。

## 🎮 在线访问

部署后访问：`https://你的用户名.github.io/heimlich-game/`

## 🚀 部署到 GitHub Pages

### 方法一：使用 GitHub 网页（最简单）

1. 登录 [GitHub](https://github.com)
2. 点击右上角 **+** → **New repository**
3. 仓库名称填写：`heimlich-game`
4. 选择 **Public**
5. 点击 **Create repository**
6. 点击 **uploading an existing file**
7. 将本文件夹中的 `index.html` 文件拖拽上传
8. 点击 **Commit changes**
9. 进入仓库 **Settings** → **Pages**
10. 在 **Source** 中选择 `main` 分支和 `/ (root)`
11. 点击 **Save**
12. 等待 1-2 分钟，你的游戏就可以访问了！

### 你的游戏链接将是：
```
https://你的用户名.github.io/heimlich-game/
```

## 📱 游戏玩法

1. 观察当前需要完成的急救步骤
2. 从下方卡片中选择正确的步骤
3. 答对的卡片会移动到上方绿色区域
4. 完成全部 8 个步骤即可获得满分！

## 📚 包含的急救步骤

1. 🔍 发现有人呛到了
2. 🧍 站到他的身后
3. 🤗 双手环抱他的腰部
4. ✌️ 一手做剪刀状，放在肚脐上部
5. ✊ 另一手握拳放在上部，拳眼贴腹部
6. ☀️ 用另一只手包住拳头
7. 💥 快速向内上方推压
8. 🔄 重复推压直到东西出来
